CSS Keyframes

Here we're going to talk about CSS keyframes. Keyframes are a way to create CSS Animations, and we'll take a look at the options you have when using keyframes.

Properties

  • @keyframes
  • animation-name
  • animation-duration
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode
  • animation

CSS Transitions

CSS Transitions are another way for animating your code. CSS Transitions allow you to change property values smoothly over a given duration.

Properties (The most important ones)

  • background-color
  • color
  • border-radius
  • transform
  • all
