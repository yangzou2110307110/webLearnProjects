Introduction

Pascal is a general-purpose, imperative and procedural programming language intended to encourage good programming practices using structured programming and data structuring.

Pascal was designed to be a language for teaching programming, and it is still widely used in schools and universities. It is also used in many commercial applications, especially for embedded systems, and is the basis for many other languages, including:

  • Delphi
  • Object Pascal
  • Free Pascal
  • Modula-2
  • Turbo Pascal
History

Pascal was developed in 1970 by Niklaus Wirth at the Swiss Federal Institute of Technology in Zurich (ETH Zurich) as a small, efficient language intended to encourage good programming practices using structured programming and data structuring. Pascal was designed to be compiled to efficient machine code for many kinds of computers, including microprocessors, minicomputers, and mainframes.

Hello world

To get started with writing Pascal, open an Editor and write your first "Hello world" Pascal code:

PROGRAMME BEGIN WriteLn("Hello world"); END.

Compile your code and run it. You should see the following output: Hello world

Declaring variables

You can declare a variable in three ways:

VAR a: INTEGER; b: REAL; c: STRING;

You have to declare your variables before using them

First Programme

A simple programme that calculates the area of a circle:

PROGRAMME VAR r: REAL; area: REAL; BEGIN WriteLn("Enter the radius of the circle: "); ReadLn(r); area := PI * r * r; WriteLn("The area of the circle is: ", area); END.

Compile your code and run it. You should see the following output: Enter the radius of the circle: 5 The area of the circle is: 78.5398

Functions and procedures

A function is a block of code that returns a value. A procedure is a block of code that does not return a value.

A simple function that calculates the area of a circle:

FUNCTION area(r: REAL): REAL; BEGIN area := PI * r * r; END;

A simple procedure that calculates the area of a circle:

PROCEDURE area(r: REAL; VAR area: REAL); BEGIN area := PI * r * r; END;
